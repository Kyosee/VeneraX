## What's Changed

### Added
- WebDAV online comic library: read comics directly from a remote directory online; reuses your existing WebDAV settings automatically when not configured separately
- QR-code sync: config now saves and takes effect immediately after scanning, no manual save needed
- Custom desktop app icon with three built-in presets to switch between
- New "Test Connection" button in the WebDAV settings dialog

### Changed
- Added a collapsible "WebDAV Sync" group in Settings that gathers sync, logs, and comic library options; setting groups are now unified as collapsible headers with icons
- Redesigned the detail page with an edge-to-edge layout, a main area that adapts to window width, and larger, better-proportioned reading buttons on wide screens
- Renamed "Disclaimer" to "User Agreement & Disclaimer"

### Fixed
- Fixed the night-view toggle being overwritten across devices when restoring a backup (#125)
- Fixed the multitasking view showing the old app name
- Fixed clearing the reading history list also wiping per-chapter reading progress
- Fixed some images failing to load in the WebDAV comic library
- Fixed the About page description text touching the screen edges (#119)
