## 更新内容

### 新增
- iOS 通过 AltStore 安装：以 AltStore 清单形式发布，每次发版自动更新
- 首次启动新增免责声明与同意门控
- 本地漫画导出改为可后台、可续传的任务（先选目标文件夹，每部一个文件）
- 导出新增「合并为单个文件包」选项
- 文件夹导入新增支持 `.venera_comics`
- 备份导入支持进度显示、后台运行，并加固大文件导入
- 网页端：新增稍后阅读，与 App 端对齐

### 变更
- Android 下载支持后台保活；Windows 支持最小化到系统托盘

### 修复
- 修复发现页多分区页面加载失败时崩溃
- 删除已安装配置后清理其登录态与本地数据，重装从干净状态开始、不再自动登录
- 修复批量下载时漫画封面错乱
- 修复图片收藏因缓存键缺少页码导致显示错图
- 修复旧版数据库表导致稍后阅读写入/导入崩溃
- 修复以毫秒时间戳命名的备份导致备份列表/下载崩溃
- 加固后台下载保活通知的刷新逻辑
- 修复 Windows 托盘开关需重启才生效
- 网页端：修复连续阅读章节重复加载/标题不同步/方向键不滚动、历史与稍后阅读详情解析、卡片显示与继续阅读页码恢复

### 构建 / CI
- 新增 AltStore 安装清单及其生成脚本与单测；发版后自动刷新清单
- 修复新增托盘功能后 Linux 发版构建无法打包的问题：在 CI 中安装 ayatana appindicator 依赖
- 更新 README 中的免责声明措辞
