## What's Changed

### Added
- In-app update dialog now shows a localized changelog fetched from the release notes (Chinese and English).

### Changed
- Side navigation bar is now scrollable in landscape so all items and actions stay reachable on short screens.
- Dialog content now has horizontal padding for more comfortable reading.

### Fixed
- Follow Updates: reading a comic now refreshes the home-screen count badge and the follow-updates list immediately (notably on the web client).
- WebDAV sync: saving the config on a fresh device now downloads the remote backup instead of being blocked by the empty-data upload guard.
- WebDAV sync: downloading a specific backup no longer leaves the loading dialog stuck.
- Import: database files are now replaced with a short retry to avoid file-in-use failures on Windows.
- Comic import: cover and chapter files are moved (with copy fallback across volumes) to avoid antivirus ransomware false positives.
- Images: a missing local image pack no longer triggers endless retries and log spam.

**Full Changelog**: https://github.com/Kyosee/venera/compare/v2.0.7...v2.0.8
