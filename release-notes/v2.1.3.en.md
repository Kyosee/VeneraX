## What's Changed

### Added
- Settings page now supports keyword search
- Swipe left on local favorites to unfavorite
- Draggable scrollbar on long list pages for quick navigation

### Fixed
- Fix search results not refreshing after removing a favorite in search mode
- Fix backup rotation sometimes deleting the newest backup instead of the oldest
- Fix background sync being frozen by the system and missing its notification card
- Fix long-press on list card tags accidentally copying text (long-press copy now only works on the detail page)
- Fill in missing translations in settings, update dialog, and other places for non-Chinese languages
