## What's Changed

### Added
- WebDAV sync frequency now offers three modes: real-time, data-saving, and manual-only
- The number of WebDAV backups to keep is now configurable

### Changed
- More reliable cross-device sync of read / unread marks for followed updates: the more recent side wins, so marking as read on one device clears the unread mark on others

### Fixed
- Fixed a crash after importing data or syncing via WebDAV
- Fixed backups not being created after enabling automatic sync
- Fixed the "finished" mark not showing in the top-left corner after reading all chapters in continuous scroll mode
- Fixed Traditional/Simplified Chinese conversion not working in some cases
