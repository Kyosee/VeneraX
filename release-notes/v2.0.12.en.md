## What's Changed

### Added
- iOS install via AltStore: the app is published as an AltStore source that auto-updates on every release
- First-run disclaimer with a consent gate before the app can be used
- Local comic export is now a resumable background task (pick a target folder, one file per comic)
- Export option to merge a comic into a single package
- Folder import now also accepts `.venera_comics`
- Backup import now shows progress, runs in the background, and is hardened for large files
- Web PWA: read-later support, aligned with the app

### Changed
- Android keeps downloads running in the background; Windows can minimize to the system tray

### Fixed
- Fixed a crash on the explore page when a multi-section page failed to load
- Deleting an installed source now clears its saved login and data, so reinstalling starts clean without auto-login
- Fixed wrong covers when batch-downloading comics
- Fixed image favorites showing the wrong image (the cache key was missing the page)
- Fixed read-later write/import crashes caused by legacy database tables
- Fixed a backup list/download crash for backups named with a millisecond timestamp
- Hardened the background download keep-alive notification refresh
- Fixed the Windows tray switch needing a restart to take effect
- Web PWA: fixed continuous-reader chapter reload, title sync and arrow-key scrolling, history/read-later detail resolution, card labels, and resume-to-last-page

### Build / CI
- Added the AltStore source manifest with a generator script and tests; releases now refresh it automatically
- Updated the disclaimer wording in README
