## What's Changed

### Added
- Multiple source libraries: subscribe to several catalogs at once and manage them on a dedicated page
- Source library manager: add, edit, delete, reorder (priority), enable/disable, browse and install per library
- Each installed source now shows which library it came from
- Switch a source to another library's variant, with the offered version shown for each library before you confirm
- Night mode can follow the system dark mode and toggle automatically
- History and Read Later support swipe-to-delete

### Changed
- Reworked the comic source page and the home source card into a cleaner card / single-line layout
- Source updates are now bound to the library a source was installed from, so a different maintainer's catalog can no longer push the wrong update
- Comic import keeps running in the background
- Follow-update checks keep running in the background and resume across restarts

### Fixed
- Fixed single-source update still downloading the old address after a source list migration
- Improved the message when a source download address cannot be resolved, and avoided cached update checks
- Fixed night mode color and intensity not being adjustable while following the system
- Fixed an illegal setState during startup when syncing night mode

**Full Changelog**: https://github.com/Kyosee/venera/compare/v2.0.12...v2.1.0
